### Overview

Xshark is a development tool designed to remotely manage and edit your GTA map in a fast and efficient way.

Comparable to **Blender** or **Roblox Studio**, Xshark goes further by offering advanced remote control and smoother workflows.

---

### Installation

1. Download the Xshark tool.
2. Run Xshark **as administrator** (optional but recommended).
3. Make sure all dependencies are properly installed.

---

### Launching

* Administrator mode improves the execution of advanced features.
* Without admin mode, some functionalities may be limited.

---

### Map connection

To connect Xshark to your GTA map:

* Start your server or development environment.
* Launch Xshark.
* Select your target map.
* Connect remotely using the Xshark interface.

---

### Main features

* Full remote map management
* Real-time editing
* Object placement and editing
* Script management
* Performance optimization
* Built-in debugging tools

### Advantages
